# TowerDefenseUnityBuild

## Опис проєкту

Це веб-версія гри **TowerDefenseUnity**, упакована як Unity WebGL збірка. Проєкт призначений для запуску в браузері та демонстрації готової гри без додаткової установки.

## Що міститься в репозиторії

- `index.html` — головна HTML-сторінка, яка завантажує Unity WebGL збірку.
- `Build/` — скомпільовані ресурси гри:
  - `Build.framework.js`
  - `Build.loader.js`
  - `Build.data`
  - `Build.wasm`
- `TemplateData/` — стилі та елементи UI для завантажувального екрану та кнопок.

## Як запускати

1. Відкрийте `index.html` у браузері, або
2. Запустіть локальний сервер у папці проєкту, наприклад через Python:

```bash
python3 -m http.server 8000
```

Потім відкрийте `http://localhost:8000`.

## Особливості

- Запуск гри через WebGL
- Підтримка повноекранного режиму
- Готова статична збірка для публікації на GitHub Pages, Netlify або будь-якому static host

## Примітка

Цей репозиторій містить готову збірку, а не вихідний код Unity-проєкту.
